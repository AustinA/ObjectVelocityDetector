# ObjectVelocityDetector

Performs sensor fusion using a VLP-16 lidar and a BlackFly camera to determine the velocities of detected objects from TensorFlow.

Package requirements:
-opencv-dev
-libpython3-dev
-libpython
-python3-dev
-pcl 1.8x

Pip3 requirements:
-python-opencv
-tensorflow
-numpy
-Cython
-contextlib2
-pillow
-lxml
-jupyter
-matplotlib
